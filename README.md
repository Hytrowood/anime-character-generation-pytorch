# Anime Character Generation using Deep Generative Models

This project explores deep generative models for anime face generation using PyTorch.

The project implements and compares three neural network architectures:

- AutoEncoder (AE)
- Variational AutoEncoder (VAE)
- Deep Convolutional GAN (DCGAN)

## Project Overview

The objective is to learn latent representations of anime face images and generate realistic new anime characters from learned distributions.

The project investigates:

- Image reconstruction using AutoEncoder
- Probabilistic latent space learning using VAE
- Adversarial image generation using DCGAN
- Comparison of generated image quality across models
